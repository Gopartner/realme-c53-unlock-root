Key Observations:
- A/B slot layout for critical partitions
- boot_a/b have NO ramdisk (kernel + DTB only)
- init_boot_a/b contain the device-specific ramdisk
- miscdata stores the bootloader unlock state
- Unisoc T612 (ums9230) architecture
